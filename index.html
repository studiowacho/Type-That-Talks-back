Every pairing on this site resolves to the same three roles — display, body, mono — plus one shared scale. Swap the families, keep the system. That's the whole trick: typography as tokens, not per-screen decisions.

The token contract

/* roles — swap families here, nothing else moves */
:root {
  --font-display: "Fraunces", Georgia, serif;
  --font-body: "Inter", system-ui, sans-serif;
  --font-mono: "JetBrains Mono", monospace;

  /* one scale, 1.4 ratio, shared by every pairing */
  --text-xs: 0.75rem;   --text-sm: 0.875rem;
  --text-md: 1rem;      --text-xl: 1.25rem;
  --text-2xl: 1.75rem;  --text-3xl: 2.5rem;

  /* leading: tight for display, loose for body */
  --leading-display: 1.15;
  --leading-body: 1.6;
}
